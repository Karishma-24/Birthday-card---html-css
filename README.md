# 🎂 Birthday Card Website

A fun and interactive birthday card website built using **HTML** and **CSS**. This project was created as part of my web development learning journey.

## ✨ Features

- 🎉 Personalized birthday greeting
- 🖼️ Circular profile image
- 🎁 Interactive gift cards that reveal GIFs on hover
- 🌈 Gradient background
- 💖 Responsive and clean layout
- ✨ Google Fonts integration

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts

## 📁 Project Structure

```
Birthday-card---html-css/
│
├── index.html
├── styles.css
├── README.md
└── images/
```

## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/Karishma-24/Birthday-card---html-css.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

That's it!
